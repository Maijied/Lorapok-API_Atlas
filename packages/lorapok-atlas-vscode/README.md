# Lorapok Atlas — VS Code Extension

> Browse, search, and insert code snippets from 2100+ free open-source APIs directly in VS Code.

[![VS Marketplace](https://img.shields.io/visual-studio-marketplace/v/lorapok.lorapok-atlas?style=flat-square&color=38bdf8&label=VS%20Marketplace)](https://marketplace.visualstudio.com/items?itemName=lorapok.lorapok-atlas)
[![License: MIT](https://img.shields.io/badge/License-MIT-34d399?style=flat-square)](../../LICENSE)

## Features

- 🔍 **Search 2100+ APIs** — by name, description, or category
- 📂 **Filter by category** — 34 categories including AI, Weather, Crypto, Privacy & Anonymity
- ⬆ **Insert into editor** — paste JavaScript, Python, or cURL snippet at cursor position
- ⎘ **Copy to clipboard** — one click copy
- 🔓 **Auth badges** — instantly see if an API is free, requires a key, or OAuth
- 🌙 **Dark themed** — matches VS Code's dark theme

## Usage

1. Click the 🐛 **Lorapok Atlas** icon in the Activity Bar (left sidebar)
2. Search for an API
3. Click any result to see details and code snippets
4. Click **⬆ Insert into Editor** to paste the snippet at your cursor

## Install

**From VS Code Marketplace:**
1. Open VS Code
2. Press `Ctrl+P` / `Cmd+P`
3. Type `ext install lorapok.lorapok-atlas`

**Or search** "Lorapok Atlas" in the Extensions panel.

## Commands

| Command | Description |
|---|---|
| `Lorapok Atlas: Open API Directory` | Focus the sidebar panel |
| `Lorapok Atlas: Insert API Snippet` | Quick insert from command palette |
