📖 How to Use Lorapok Atlas

    
Your zero-config API playground in Firefox

    

      

        
1

        
Browse & Filter
Click any category in the left sidebar to filter APIs. Use the search bar to find by name, description, or category. Filter by auth type: Free 🔓 / Key 🗝 / OAuth 🔑

      

      

        
2

        
View API Details
Click any card to open the detail modal. You'll see the endpoint URL, description, HTTP method, and auth requirements.

      

      

        
3

        
Copy Code Snippets
Switch between JavaScript, Python, and cURL tabs. Click "⎘ Copy Snippet" to copy the ready-to-use code to your clipboard.

      

      

        
4

        
Run Live Tests
Click "▶ Run Test" button below the URL to expand the test panel. Add custom headers if needed, then hit Run. The extension tries direct fetch first, then falls back through CORS proxies automatically.

      

      

        
5

        
Get API Keys
If an API requires a key, click "🔑 Get API Key" to open the signup page in a new tab.

      

    

    
💡 Tip: Click the 🐛 larva icon in the header to toggle the category sidebar for more grid space.
