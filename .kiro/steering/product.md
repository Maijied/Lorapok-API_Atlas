# Lorapok API Atlas

A web dashboard for exploring and live-testing 100+ curated free/open-source APIs. Users can browse APIs by category, run live requests, view visualized responses (JSON, images, video, HTML), copy code snippets, and manage API keys.

## Key Features
- Live API testing with real-time response visualization (JSON, images, video, HTML previews, binary data)
- Code snippet generation (cURL, JavaScript, Python, Go)
- API key management backed by Firebase Firestore (synced across devices when signed in)
- Google OAuth authentication via Firebase
- Lorapok larva mascot providing status feedback during requests
- 20+ API categories with emoji icons

## Target Users
Developers who want a zero-config sandbox to explore and test public APIs.

## Deployment
Hosted on GitHub Pages at `/Lorapok-API_Atlas/`. Auto-deployed via GitHub Actions on push to `main`.
